# Modelo de Portfólio

Projeto de site de portfólio desenvolvido com **HTML5 semântico** e **CSS3**, criado como prática de front-end no SCTEC / Senai SC / LAB365 (20h).

🔗 **Acesse online:**  
https://cristina-freitas.github.io/site/

---

## 📌 Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de aplicar fundamentos sólidos de desenvolvimento front-end, priorizando organização estrutural, boas práticas e separação de responsabilidades.

Foram aplicados conceitos de:

- Estruturação semântica com HTML5
- Separação entre estrutura (HTML) e estilo (CSS)
- Navegação interna por âncoras
- Construção adequada de tabelas
- Criação de formulários com validação básica
- Organização clara de layout

O foco foi construir uma base técnica correta e escalável.

---

## 🛠 Tecnologias Utilizadas

- HTML5
- CSS3

## 🧱 Estrutura Semântica Utilizada

O projeto aplica corretamente as seguintes tags:

- `header`
- `nav`

- `article`
- `aside`

Essa organização demonstra entendimento de arquitetura estrutural, acessibilidade e boas práticas para SEO.

- Nome (obrigatório)
- E-mail (obrigatório)
- Telefone
- Assunto (select)
- Mensagem (obrigatória)
- Checkbox para newsletter

> Observação: O formulário ainda não está conectado a backend. O foco deste projeto foi a estrutura front-end.
